1.Sistem Kasir
2.Deskripsi proyek
Proyek ini bertujuan untuk membuat aplikasi kasir berbasis web yang dapat mempermudah proses transaksi penjualan, pembelian dari supplier, manajemen produk, karyawan, dan laporan keuangan. Sistem ini ditujukan untuk toko ritel kecil hingga menengah.
3.Tujuan proyek
Mengotomatisasi proses penjualan dan pembelian.
Mempermudah pengelolaan stok produk.
Menyediakan laporan penjualan dan pembelian secara real-time.
Menyediakan manajemen user dengan role admin dan kasir.
Meningkatkan akurasi pencatatan transaksi dan mengurangi kesalahan manusia.
4.Fitur Utama
5.Spesifikasi Teknis
Frontend: HTML, CSS, JavaScript, Webix UI
Backend: Python (FastAPI)
Database: MongoDB
Validasi & Sanitasi: Pydantic
Deployment: Server lokal / cloud
Hashing : bycrypt
6.Alur sistem
Flowchart
User login sesuai role (admin/kasir)
Admin dapat mengelola karyawan, produk, dan laporan
Kasir dapat melakukan transaksi penjualan
Sistem otomatis mengupdate stok dan generate laporan
Data tersimpan di database MongoDB
